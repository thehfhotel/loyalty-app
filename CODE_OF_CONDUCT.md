# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, give and accept constructive criticism. Harassment, personal attacks, and discriminatory comments are not tolerated in issues, pull requests, discussions, or any other project space.

## Reporting

Conduct issues can be reported by emailing the maintainers (see the address in [SECURITY.md](./SECURITY.md)). Reports are kept confidential. Maintainers will respond within 7 business days.

The full Contributor Covenant text — including enforcement guidelines and the four-tier consequence ladder — is at the link above.
